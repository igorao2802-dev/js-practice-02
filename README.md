# Практическое задание 2. Тема: Условные операторы и логика приложения в JavaScript (if/else, else if, тернарный оператор, switch), базовая валидация пользовательского ввода

## Проект: мини-приложение «личный кабинет покупателя». 

### Описание проекта: учебный проект на одной странице, который имитирует реальный «личный кабинет покупателя» с функционалом: вход/валидация, расчёт скидки, конвертер валют и мини-квиз.

### Используемые технологии: 
- **HTML** — структура страницы;
- **CSS** — оформление и стилизация;
- **JavaScript** — логика и функционал.
