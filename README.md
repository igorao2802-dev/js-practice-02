# Практическое задание 2. Тема: Условные операторы и логика приложения в JavaScript (if/else, else if, тернарный оператор, switch), базовая валидация пользовательского ввода

## Проект:

мини-приложение «личный кабинет покупателя».

## Описание проекта:

учебный проект на одной странице, который имитирует реальный «личный кабинет покупателя» с функционалом: вход/валидация, расчёт скидки, конвертер валют и мини-квиз.

## Используемые технологии:

- **HTML** — структура страницы;
- **CSS** — оформление и стилизация;
- **JavaScript** — логика и функционал.

## Как запускать:

### Вариант 1 - без локальной копии (через ZIP)

1. Открыть репозиторий:  
   [js-practice-02](https://github.com/igorao2802-dev/js-practice-02/)
2. Выбрать **Code** :arrow_right:; **Download ZIP** :arrow_right: **Укаказать место сохранения архива** :file_folder:.
3. Распаковать архив и открыть папку проекта в **VS Code**.
4. Установить расширение **Live Server** (если оно отсутствует).
5. Открыть файл :page_facing_up: "index.html":
   - через меню: **File** :arrow_right: **Open File…** :arrow_right: `index.html`
   - затем выбрать **Open with Live Server**;
   - или через нижнюю панель VS Code — кнопку <kbd>**Go Live**</kbd>

--

### Вариант 2 - с локальной копией (через Git)

#### Способ 1: через терминал

1. Клонировать репозиторий проекта:

   ```bash
   git clone https://github.com/igorao2802-dev/js-practice-02/

   ```

2. Перейти в папку проекта:

   ```bash
   cd js-practice-02

   ```

3. Открыть проект в VS Code:

   ```bash
   code

   ```

4. Установить расширение **Live Server** (если оно отсутствует).

5. Запустить проект:

- открыть `index.html`;
- выбрать `Open with Live Server`.

--

#### Способ 2: через меню VS Code

1. Открыть VS Code.

2. Выбрать **View** :arrow_right: **Terminal**.

3. В терминале выполнить:

   ```bash
   git clone https://github.com/igorao2802-dev/js-practice-02/

   ```

4. В меню выбрать **File** :arrow_right: **Open Folder…** и указать папку :file_folder: **js-practice-02**.

5. Открыть `index.html` и запустить через **Open with Live Serve**r.
